## 🌟 About the Project

This is a **content-based movie recommendation system** that suggests movies based on **genre and language similarity**.  
Built using **Python, NLP, and cosine similarity**, the project demonstrates how machine learning can be applied to real-world problems in a simple and effective way.

---

## ⚙️ How It Works

🔹 Loads and merges movie datasets  
🔹 Cleans and processes the data  
🔹 Extracts important features (genres, keywords, language)  
🔹 Creates a combined **tags** column  
🔹 Converts text into vectors using NLP  
🔹 Calculates similarity using cosine similarity  
🔹 Recommends movies based on input  

---

## 🎯 Features

✨ Recommend movies based on a selected movie  
✨ Filter movies using **genre & language**  
✨ Fast and efficient recommendation  
✨ Clean and modular code structure
